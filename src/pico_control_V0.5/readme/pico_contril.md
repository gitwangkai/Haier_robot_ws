🔌 一、 准备工作
在开始之前，请确保：
硬件连接：机械臂已通过USB串口线正确连接到您的电脑上。
网络环境：您的电脑和Pico VR一体机已连接到 同一个Wi-Fi网络。
VR设备：Pico VR一体机已开机，并已运行配套的VR服务端应用。
💡 提示：找到pico的IP地址，请记下它，启动VR服务端应用后。
🚀 二、 启动流程
请按照以下步骤启动控制程序：
运行程序：在电脑上打开一个终端，运行主控制脚本。
输入IP地址：
程序会提示您输入VR服务端的IP地址。
请输入您在VR设备上看到的IP地址，然后按回车。
等待连接：
当您看到 ✅ VR服务端连接成功！机器人功能已锁定。 的提示时，表示启动成功。
此时，机器人所有功能都处于 锁定状态，不会响应任何移动指令，以确保安全。
🎮 三、 核心操作
1. 强制初始化 (解锁功能的唯一方式！)
⚠️ 重要：为了安全，连接成功后 必须 先手动初始化机械臂，才能解锁其他功能。
操作：
按一下 左手柄的Y键，初始化 左臂。
按一下 右手柄的B键，初始化 右臂。
效果：
对应的机械臂会以一个较慢的速度（5秒），平稳地移动到安全的初始“小臂平行地面”姿态。
终端会显示 ✅ 左/右臂已初始化，功能已解锁。
解锁条件：
手臂/夹爪：对应手臂初始化后即可使用。
底盘移动：必须双臂均已初始化 后才能解锁。
2. 遥操作 (解锁后可用)
在对应功能解锁后，您可以自由操控：
移动手臂：
左臂：按住 左手柄的主按键 (X)，移动您的手。
右臂：按住 右手柄的主按键 (A)，移动您的手。
控制夹爪：
左夹爪：扣动 左手柄的扳机 (Trigger)。
右夹爪：扣动 右手柄的扳机 (Trigger)。
控制底盘移动 (双臂均初始化后)：
使用 右手柄的摇杆 (Joystick)。
前/后：前进/后退
左/右：原地转向
3. 姿态复位
在操作过程中，您可以随时让已初始化的手臂恢复到初始姿态。
操作：按一下手柄的 次要按键 (B键/Y键)。
效果：对应的手臂会缓慢地回到初始姿态。
🛑 四、 安全功能：紧急停止
安全第一！ 急停功能拥有最高优先级，不受初始化锁的限制。
触发急停：
同时按住 右手柄的 A键 + B键。
急停效果：
底盘立即停止。
双臂立即卸力：机械臂立刻变“软”，失去所有力气。
从急停中恢复：
松开 A键 和 B键。
此时，机器人将 重新回到功能锁定状态。
您 必须 再次按照 第 1 步 (强制初始化) 的流程，按B/Y键重新初始化机械臂，才能恢复遥操作。这确保了每次急停后的恢复都是安全的。
🛠️ 五、 常见问题排查 (Troubleshooting)
问题：连接成功后，按A键或动摇杆没反应？
解答：这是正常的安全锁定。请先按 B键/Y键 完成手臂的 强制初始化 以解锁功能。
问题：程序提示 连接超时 或 连接被拒绝？
解答：请检查：
VR一体机上的服务端App是否已正确运行？
您输入的IP地址是否正确？
电脑和VR一体机是否在同一个Wi-Fi下？
问题：程序提示 串口发送失败 或 No such file or directory？
解答：这表示程序没有找到您的机器人串口。
请在电脑终端中运行 ls /dev/tty* 命令，找到您机器人的正确端口号（通常是 /dev/ttyUSB0、/dev/ttyUSB1- ̗̀ a ̖́- 等）。
打开主控制脚本文件，在文件最顶部的“全局配置常量”部分，修改 LEFT_ARM_PORT 和 RIGHT_ARM_PORT 的值为您找到的正确端口号。
