### 核心函数（在 Board.py 中）

#   high: 最大位置 (0-1000)

low, high = getBusServoAngleLimit(id)

## 📊 常用行程设置

| 用途 | 最小值 | 最大值 | 说明 |
|------|--------|--------|------|
| **全行程** | 0 | 1000 | 出厂默认，240度 |
| **限制90%** | 50 | 950 | 略微限制 |
| **限制80%** | 100 | 900 | 常用安全范围 |
| **限制60%** | 200 | 800 | 较大限制 |
| **限制50%** | 250 | 750 | 中等行程 |
| **中心±30°** | 450 | 550 | 小范围摆动 |

**位置与角度的关系：**
- position = 0 → 0°
- position = 500 → 120° (中位)
- position = 1000 → 240°
- 公式: `角度 = position × 0.24`
