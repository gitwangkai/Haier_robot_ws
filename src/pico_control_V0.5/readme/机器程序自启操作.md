## 修改picoIP地址:

```
echo "192.168.0.104" > /home/aidlux/pico/pico_control_V0.5/vr_server_config.txt
```

修改ip地址为对应的pico的IP地址，直接粘贴在终端执行。**每次修改完要使新的IP地址生效需要重启服务**

```
sudo systemctl restart ros_launch.service
```

一般情况下只会用到这两条命令





手动停止已自启动的服务:

```
sudo systemctl stop ros_launch.service
```

手动启动已停止的的服务:

```
sudo systemctl start ros_launch.service
```

禁用程序开机自启动：

```
sudo systemctl disable ros_launch.service
```

启用程序开机自启动：

```
sudo systemctl enable ros_launch.service
```

重新启动开机自启动的服务:

```
sudo systemctl restart ros_launch.service
```



手动启动pico遥操（在禁用自启动或手动停止服务后用手动启动遥操）：

```
sh /home/aidlux/start_pico.sh
```



查看日志:

```
sudo journalctl -u ros_launch.service -f
```
